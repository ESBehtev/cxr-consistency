# cxr-consistency

Проект посвящён задаче мультимодальной проверки согласованности рентгеновских снимков грудной клетки (Chest X-ray, CXR) и текстовых радиологических заключений.

Модель получает на вход пару:

- рентгеновский снимок;
- текстовое заключение.

На выходе модель определяет:

- label = 1 — снимок и заключение согласованы;
- label = 0 — снимок и заключение противоречат друг другу.

В отличие от задач генерации медицинских отчётов, данная работа рассматривает задачу multimodal consistency checking — обнаружение семантических несоответствий между визуальной и текстовой модальностями.

В рамках исследования сравниваются визуальные энкодеры ConvNeXt, ViT и DeiT в сочетании с медицинским текстовым энкодером CXR-BERT на специально сформированном наборе сложных негативных примеров.

## Постановка задачи

Целью работы является автоматическое определение соответствия между рентгеновским изображением и текстовым описанием исследования.

Особое внимание уделяется клинически реалистичным ошибкам:

- отрицаниям патологий;
- конфликтам локализации;
- временным несоответствиям;
- подмене патологий;
- частичным противоречиям между изображением и текстом.

Задача формулируется как бинарная классификация пары (image, report).

## Используемые данные

В работе используются данные MIMIC-CXR и набор текстовых заключений CXR Findings.

Основные датасеты:

- MIMIC-CXR Dataset:
  https://www.kaggle.com/datasets/simhadrisadaram/mimic-cxr-dataset

- CXR Findings Dataset:
  https://huggingface.co/datasets/erjui/csrrg_findings

Разделение данных выполняется на уровне пациентов (patient-level split), что предотвращает утечку информации между обучающей и тестовой выборками.

## Архитектура модели

Используется двухпоточная мультимодальная архитектура.

### Визуальные энкодеры

- ConvNeXt Tiny
- DeiT Base
- ViT Base

Реализация основана на библиотеке timm.

### Текстовый энкодер

Используется специализированная медицинская модель:

- microsoft/BiomedVLP-CXR-BERT-specialized

Текстовый энкодер обучается совместно с визуальной частью модели.

### Слияние модальностей

После получения визуального вектора v и текстового вектора t используется схема объединения:

text concat(v, t, |v − t|, v * t) 

Полученное представление подаётся в многослойный классификатор, который выдаёт вероятность согласованности пары.

## Типы сложных негативных примеров

В финальной версии используются следующие типы негативных пар:

- pathology_matched_report — заключение из другого исследования с похожими патологиями;
- distorted_negation — изменение отрицаний патологий;
- laterality_conflict — конфликт левой и правой стороны;
- temporal_mismatch — нарушение временного контекста;
- pathology_semantic_swap — замена патологии на другую;
- partial_mismatch — частичное изменение одного из наблюдений.

Наиболее сложным типом негативных примеров является pathology_matched_report.

## Итоговые результаты

Финальные эксперименты выполнялись на полном наборе сложных пар с использованием CXR-BERT и ранней остановки по метрике ROC-AUC на валидационной выборке.

| Модель | Лучшая эпоха | ROC-AUC | F1 | AUC для pathology_matched_report |
|----------|----------:|----------:|----------:|----------:|
| ConvNeXt Tiny + CXR-BERT | 8 | 0.9228 | 0.7378 | 0.8305 |
| DeiT Base + CXR-BERT | 5 | 0.9156 | 0.7244 | 0.8159 |
| ViT Base + CXR-BERT | 7 | 0.9072 | 0.7142 | 0.8077 |

## Основные выводы

- ConvNeXt Tiny показал наилучшие результаты среди всех исследованных архитектур.
- DeiT Base оказался лучшим трансформерным визуальным энкодером.
- ViT Base служит сильной базовой трансформерной моделью, однако уступает DeiT и ConvNeXt.
- Наиболее сложным типом негативных примеров является pathology_matched_report.
- Использование CXR-BERT существенно улучшает качество выявления клинически реалистичных несоответствий.
- После достижения оптимальной эпохи наблюдается переобучение и смещение оптимального порога классификации, поэтому для финальной оценки используются результаты ранней остановки.
